はじめまして。平成5年生まれの28歳エンジニアのジョウニニテッと申します。社会人歴（IT業界）7年目です。来日後、現在会社にて 3 年間、J2EE 技術を利用した Web 系システムの開発を勤務してきました 。 Java の特徴であるオブジェクト指向を有効利用して、再利用性の高いソフトウェアを目指して開発設計および プログラミング、試験にスムーズに対応できる強みがあります。

私の詳細について

私は大学卒業後、ITエンジニアとしてミャンマーで４年間勤務しました。この後、ミャンマーではなく、日本でもSEのキャリアを磨きたいと思い、2019年1月から今の会社に入社しました。入社した後から今まで様々なシステム開発のプロジェクトにかかわってきました。私はプログラミングに関するスキルに自信があります。JavaやPHPやJavascriptはもちろん、Rubyに関しても開発経験がありますので、WEBシステムならバックエンドだけではなく、フロントエンドも対応できます。日本人とコミュニケーション能力及び思考・分析能力には自信があり、最新技術を使って様々なシステムを開発し、役に立てる技術者になりたいです。

特技　‐　ソフトウェア開発の仕事をよくできる事です。（フロントエンド且つバックエンド）
趣味　‐　運動（ジョギング）、映画を見ること

勤務先を東京とさせていただければ幸いです。
